> 让兔兔可以做一个简易骰娘

    - 进行一次公开roll
        - 发送`roll [x]d[y][+z]` 或者 `.r [x]d[y][+z]` (注意前面那个点)
        - 例如`roll 10d12`
        - 例如`.r 2d6`
    
    - 进行一次秘密roll
        - 发送`sroll [x]d[y][+z]` 或者 `.rh [x]d[y][+z]` (注意前面那个点)
        - 例如`sroll 10d12`
        - 例如`.rh 2d6`
        - 掷骰结果会以私聊消息发送给你，你需要加兔兔好友，兔兔暂不支持临时消息
